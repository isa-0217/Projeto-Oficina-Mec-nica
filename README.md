
Este projeto representa um banco de dados para gerenciamento de uma oficina.

Entidades:
- Cliente
- Veículo
- Ordem de Serviço
- Mecânico
- Serviço
- Peça

Funcionalidades:
- Cadastro de clientes e veículos
- Controle de ordens de serviço
- Associação de serviços e peças

Queries:
- SELECT simples
- WHERE
- JOIN
- GROUP BY + HAVING
- ORDER BY
